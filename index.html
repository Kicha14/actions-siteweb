Audi & le Groupe B

Mini‑site réalisé dans le cadre du projet GitHub Actions / Nginx

Présentation

Le Groupe B représente l’époque la plus extrême du rallye automobile. Audi y a marqué l’histoire grâce à sa technologie Quattro et ses voitures légendaires comme la Sport Quattro S1.

Les modèles emblématiques

  • Audi Quattro A1 / A2 – Les débuts de la transmission intégrale en rallye.
  • Audi Sport Quattro – Empattement raccourci, matériaux composites.
  • Audi Sport Quattro S1 – Plus de 500 ch, aérodynamique extrême.

Conception du projet

Ce site web a été réalisé dans le cadre d’un projet visant à :

  • Créer un dépôt GitHub contenant un site web simple.
  • Cloner le dépôt sur une machine Linux (Debian).
  • Configurer un self‑hosted runner GitHub Actions.
  • Déployer le site via un serveur Nginx.
  • Utiliser uniquement un fichier index.html pour la structure et le style.

Le site est volontairement léger, autonome et compatible avec GitHub Pages ou un déploiement automatisé via GitHub Actions.
